# Структура экспериментов

## Общая информация

- **Реальное время обучения:** 24 часа на запуск (фиксировано)
- **Количество запусков:** 3 на алгоритм/эвристику
- **Параллелизация:** 3 VM
- **Общее время:** ~9 дней

## Структура папок

```
experiments/
├── training/                    # Обучение
│   ├── ppo/                     # PPO (3 запуска)
│   │   ├── run1.json           # seed=42
│   │   ├── run2.json           # seed=123
│   │   └── run3.json           # seed=456
│   ├── sac/                     # SAC (3 запуска)
│   │   ├── run1.json           # seed=42, 1600 шагов
│   │   ├── run2.json           # seed=123, 1600 шагов
│   │   └── run3.json           # seed=456, 1600 шагов
│   ├── td3/                     # TD3 (3 запуска)
│   │   ├── run1.json           # seed=42, 960 шагов
│   │   ├── run2.json           # seed=123, 960 шагов
│   │   └── run3.json           # seed=456, 960 шагов
│   └── baseline/                # Baseline эвристики
│       ├── static/              # Static (3 запуска)
│       │   ├── run1.json
│       │   ├── run2.json
│       │   └── run3.json
│       └── greedy_latency/      # Greedy latency (3 запуска)
│           ├── run1.json
│           ├── run2.json
│           └── run3.json
└── reconciliation/              # Reconciliation
    ├── profile_a/               # Профиль A (стабильность)
    │   └── README.md
    └── profile_b/               # Профиль B (обобщение)
        └── README.md
```

## План выполнения

### Фаза 1: Baseline (1 день)
- Static: 3 запуска параллельно на 3 VM = 1 день
- Greedy latency: 3 запуска параллельно на 3 VM = 1 день

### Фаза 2: Обучение RL-алгоритмов (3 дня)
- PPO: 3 запуска параллельно на 3 VM = 1 день
- SAC: 3 запуска параллельно на 3 VM = 1 день
- TD3: 3 запуска параллельно на 3 VM = 1 день

### Фаза 3: Reconciliation - Стабильность (3 дня)
- 9 моделей × Профиль A
- По 3 параллельно на 3 VM
- 3 батча × 24 часа = 3 дня

### Фаза 4: Reconciliation - Обобщение (3 дня)
- 9 моделей × Профиль B
- По 3 параллельно на 3 VM
- 3 батча × 24 часа = 3 дня

### Фаза 5: Baseline Reconciliation (2 дня)
- Static: Профиль A и B = 1 день
- Greedy latency: Профиль A и B = 1 день

**Итого:** ~9 дней

## Итоговая статистика

- **Обучение:** 15 экспериментов (3 алгоритма × 3 + 2 эвристики × 3)
- **Reconciliation:** 20 экспериментов (9 моделей × 2 профиля + 2 эвристики × 2 профиля)
- **Всего:** 35 экспериментов
